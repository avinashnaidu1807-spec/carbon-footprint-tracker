# 🌍 EcoTrack — Carbon Footprint Awareness Platform

A beautiful, fully interactive web platform that helps individuals understand, track, and reduce their carbon footprint through personalized insights and simple daily actions.

![EcoTrack Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

- **🧮 Carbon Calculator** — 4-step questionnaire covering Transport, Energy, Food & Lifestyle
- **📊 Interactive Dashboard** — Chart.js powered donut, bar & line charts
- **💡 Personalized Tips** — 13 filterable action tips with CO₂ savings estimates
- **📈 Impact Tracker** — Log eco-friendly actions, track streaks & earn achievements
- **🤝 Green Pledge System** — Commit to sustainability goals with impact tracking
- **🎨 Particle Background** — Animated canvas with connected nodes
- **💾 Persistent Data** — All progress saved via localStorage
- **📱 Fully Responsive** — Optimized for mobile, tablet & desktop

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/avinashnaidu1807-spec/carbon-footprint-tracker.git
   cd carbon-footprint-tracker
   ```

2. **Open in browser**
   Simply open `index.html` in your browser, or use a local server:
   ```bash
   npx serve
   ```

3. **Start tracking!**
   Use the calculator, log actions, and take pledges.
